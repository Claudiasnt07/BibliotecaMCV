# Biblioteca MVC

Este proyecto es un programa de Java que simula una pequeña biblioteca.  
Permite crear libros, usuarios y realizar préstamos.

El proyecto está hecho usando el patrón MVC.
## Estructura del proyecto

biblioteca-mvc/

src/

model/  
- Libro.java  
- Usuario.java  
- Prestamo.java  
- Genero.java  
- EstadoLibro.java  

controller/  
- GestorBiblioteca.java  

view/  
- Consola.java  

Main.java  

## Funciones del programa

- Crear libros
- Crear usuarios
- Prestar libros
- Mostrar libros
- Mostrar usuarios y sus préstamos

Un usuario puede tener máximo 3 libros prestados.
